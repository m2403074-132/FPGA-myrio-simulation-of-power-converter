# FPGA-myrio-simulation-of-power-converter
El código desarrollado en LabVIEW FPGA implementa los modelos discretizados de los convertidores Boost, Buck y Boost-Buck mediante el método de Euler. Optimiza operaciones aritméticas para reducir latencia en ciclos de reloj, usa formato fijo UQ10.32 y genera PWM, permitiendo simulaciones HIL en tiempo real sobre la plataforma NI MyRIO 1900.
